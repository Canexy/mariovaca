Memoria sobre el artículo VoIP.

El artículo explica cómo configurar un servidor SIP para usar servicios VoIP en las redes de Movistar y O2. Detalla los parámetros necesarios, como direcciones de servidores SIP (ej. sip.movistar.es), números de usuario (asociados al teléfono fijo), contraseñas específicas y puertos (5060/5061). También menciona la importancia de ajustar códecs de audio (G.711, G.729) y protocolos (UDP/TCP) para optimizar la calidad de las llamadas.

Incluye pasos para acceder a la configuración de routers compatibles, habilitar SIP ALG (Application Layer Gateway) y resolver problemas comunes, como bloqueos de puertos o autenticación incorrecta. Además, destaca diferencias entre operadoras: Movistar suele requerir autenticación con número de teléfono, mientras O2 puede usar credenciales personalizadas.

Finalmente, advierte que estas configuraciones son exclusivas para clientes de Movistar y O2, requiriendo acceso a sus redes y datos de autenticación. No puedo replicar los pasos descritos, ya que no dispongo de servicios contratados con estas operadoras.
