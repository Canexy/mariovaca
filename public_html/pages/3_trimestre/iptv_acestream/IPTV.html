Memoria sobre IPTV y mi interpretación personal usando Acestream.

Este documento explica el proceso que sigo para organizar y reproducir canales de televisión utilizando enlaces acestream://. Utilizo estos enlaces para crear listas M3U personalizadas, organizadas por categorías, tipo de canal y EPG, que luego subo a Pastebin en formato raw para facilitar su acceso remoto. En una Smart TV, a través de un Fire Stick con la aplicación TiviMate y el motor de Acestream instalado, importo dicha lista desde Pastebin. Esto me permite ver todos los canales estructurados tal como los configuré, y actualizar la lista fácilmente modificando el contenido en Pastebin, sin necesidad de tocar la configuración del dispositivo.

  1. Generación de enlaces y organización por categorías.

Acestream es una tecnología basada en P2P que permite la transmisión de contenidos en streaming a través de enlaces con el prefijo acestream://, seguidos de un identificador único. Este protocolo se utiliza principalmente para la emisión de eventos en directo y canales de televisión en alta calidad.

Cuando se dispone de múltiples enlaces Acestream, se puede organizar su reproducción mediante la creación de una lista M3U, un formato de texto estándar para listas de reproducción. En estas listas se define el nombre del canal, tipo, categoría, el propio enlace y, opcionalmente, un identificador EPG.

Ejemplo de entrada en M3U:

#EXTM3U

#EXTINF:-1 tvg-logo="https://raw.githubusercontent.com/tv-logo/tv-logos/main/countries/spain/dazn-laliga-es.png" group-title="Fútbol 1ª División" tvg-id="DAZN LaLiga", DAZN LaLiga

acestream://110d441ddc9713a7452588770d2bc85504672f47

  1. Publicación de la lista M3U.

Para facilitar el acceso remoto y mantener centralizada la configuración, la lista M3U se sube a un servicio como Pastebin, usando el modo raw (texto plano sin formato HTML). Esto permite a los reproductores externos importar la lista como una URL directa.

  1. Reproducción en Smart TV con Fire Stick.

En una Smart TV (en este caso con Fire Stick de Amazon) se instalan dos componentes clave:

  • TiviMate: una aplicación IPTV avanzada que permite importar listas M3U, agrupar canales por categorías, mostrar la guía EPG, y administrar múltiples listas.
  • Motor de Acestream para Android (aceengine): necesario para que los enlaces acestream:// puedan ser convertidos y reproducidos localmente como flujo de vídeo.

TiviMate permite añadir una fuente remota (como la URL raw del pastebin), e interpreta toda la estructura de la lista M3U, incluyendo categorías y nombres personalizados. Cada vez que se desea modificar la parrilla, basta con editar el contenido de Pastebin y actualizar la lista desde la propia app, sin necesidad de reconfigurar el dispositivo.

El artículo de Genbeta describe cómo usar VLC como reproductor de TDT o IPTV, cargando listas M3U desde fuentes públicas, incluso grabando el contenido. Aunque es funcional, existen diferencias clave respecto al enfoque que he explicado.

Mientras que VLC ofrece una solución accesible y versátil para reproducir listas IPTV, el enfoque basado en Acestream, TiviMate y Pastebin ofrece un sistema mucho más automatizado, personalizable y cómodo para el uso en Smart TVs. Además, permite aprovechar la calidad de las transmisiones P2P sin depender de listas IPTV públicas tradicionales, siendo especialmente útil para usuarios más avanzados o que requieren una experiencia tipo "decodificador".
