Enrutamiento estático.

  1. ¿Qué comando se puede usar para configurar una ruta estática que es equivalente a la ruta por defecto?

El comando sería equivalente a route add -net 0.0.0.0 netmask 0.0.0.0 gw 195.11.14.1 dev eth0.

  1. ¿Qué sucede si teniendo rutas por defecto en r1 y r2, hacemos en pc1 ping a 200.1.1.7? ¿Por qué?

No podríamos hacer ping ya que el destino no se puede alcanzar. Eso es debido a la configuración de las rutas.

  1. ¿Cómo se arregla?

Estableciendo mediante comandos para cada nodo el gateway predeterminado.

  1. Añadir un nuevo switch y un nuevo nodo.

Así queda la configuración de lab.conf, sw1.startup y pc3.startup.

Para ahorrar tiempo, meto en la configuración del resto de elementos los comandos para añadir los gateway predeterminados.

Comprobamos que existe comunicación entre los nodos.
