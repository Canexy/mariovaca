Memoria Kathará Práctica #2

  1. Creación de un laboratorio de una red de 3 hosts conectados a un switch.

Para la creación de un laboratorio de 3 nodos y un switch, debemos crear un archivo lab.conf con la configuración del mismo. Contendrá la configuración de los nodos:

pc1[0]=A

pc2[0]=A

pc3[0]=A

Y la configuración del switch que vamos a crear, que contendrá las siguientes líneas:

sw[0]=A

sw[1]=B

sw[2]=C

Además del archivo lab.conf, crearemos 3 archivos independientes para cada nodo, con nombre pcx.startup (donde x es el número de nodo).

Cada nodo contendrá la siguiente configuración, dependiendo del número de nodo (X variará con el número del nodo nuevamente).

ifconfig eth0 10.0.0.X

Por último, crearemos otro archivo con la configuración del dispositivo switch, llamado en mi caso sw.startup, que contendrá las siguientes líneas:

brctl addbr br0

brctl addif br0 eth0

brctl addif br0 eth1

brctl addif br0 eth2

Estas líneas de configuración serán las encargadas de la creación del puente br0 y la unión de las diferentes interfaces a dicho puente.

ip link set dev br0 up

ip link set dev eth0 up

ip link set dev eth1 up

ip link set dev eth2 up

Estas otras líneas de comando serán las encargadas del levantamiento del puente creado anteriormente y de las propias interfaces.
